# Platinum Dork Scanner V14

Bu proje; yetkili ve etik güvenlik testleri kapsamında, arşivlenmiş URL’lerin toplanması,
filtrelenmesi ve dork tabanlı akıllı analizinin yapılması amacıyla geliştirilmiş
bir analiz aracıdır.

## Özellikler
- Wayback Machine üzerinden URL toplama
- Açık kaynak tehdit istihbaratı entegrasyonu
- Gürültü azaltma ve akıllı filtreleme
- Kritik, yüksek ve normal seviye sonuç ayrımı
- Grafik arayüz (Tkinter)
